F.A.Q.

Want to add a block with the most frequently asked questions? Then use the code below. Do you want to see it in action? Have a look at the FAQ page or the Elements page.

{{< faq >}}

You can load a subset of the FAQ’s by adding an index number. It will then show only one list of faqs (in the example below only the first ‘faqs’ array). This looks like this:

{{< faq 0 >}}
